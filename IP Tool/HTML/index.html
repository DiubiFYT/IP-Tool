La Subnet Mask

La Subnet Mask è un parametro in formato dotted-decimal che definisce la dimensione della sottorete o delle sottoreti a cui appartengono gli host.

Da essa si possono ricavare il numero di host assegnabili, sottoreti e reti che possono essere gestite da un router.

255.255.255.192

11111111.11111111.11111111.

Net ID

11

Subnet ID

111111

Host ID

In questo caso avremo disponibili 216 reti, 22 sottoreti e 26 hosts.
Per ricavare gli indirizzi assegnabili agli host bisognerà fare 2n-2 (-3 se si vuole escludere l'indirizzo di gateway).

Le Classi IP

Vengono usate per dividere lo spazio di indirizzamento IPv4.

Ne esistono 5 tipi, analizziamo i primi 3:

Classe A
Il primo ottetto di bits rappresenta la rete, gli altri tre gli host per ogni rete
Gli IP variano nel modo seguente: 1-127.H.H.H
La maschera di rete è 255.0.0.0 (CIDR /8)
Questi indirizzi in binario iniziano con il bit 0

Il Subnetting

Il subnetting è quella tecnica che permette di suddividere una rete in più sottoreti, in modo tale da risparmiare sugli indirizzi IP.

La parte Host ID di una rete viene suddivisa in Subnet-network ID e Host ID
