La Subnet Mask

La Subnet Mask è un parametro in formato dotted-decimal che definisce la dimensione della sottorete o delle sottoreti a cui appartengono gli host.

Da essa si possono ricavare il numero di host assegnabili, sottoreti e reti che possono essere gestite da un router.

255.255.255.192

11111111.11111111.11111111.

Net ID

11

Subnet ID

111111

Host ID

In questo caso avremo disponibili 216 reti, 22 sottoreti e 26 hosts.
Per ricavare gli indirizzi assegnabili agli host bisognerà fare 2n-2 (-3 se si vuole escludere l'indirizzo di gateway).
